Detailed Description

template<class T>
class ARK::Geometry::Polygon< T >

Class for Polygon construction. Polygons should be convex for accurate collisions as there is no Triangulation code.

Warning:
On Mac there is a conflict in this class name. You will need to use ARK::Geometry::Polygon in all cases if compiling for Mac.
