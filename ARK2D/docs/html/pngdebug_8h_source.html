00020  * png_debug[1-2]?(level, message ,arg{0-2})
00021  *   Expands to a statement (either a simple expression or a compound
00022  *   do..while(0) statement) that outputs a message with parameter
00023  *   substitution if PNG_DEBUG is defined to 2 or more.  If PNG_DEBUG

00028  *          message is preceded by 'n' tab characters (not implemented

00030  *          defined, to allow debug DLL compilation with no standard IO).
