00243   /*
00244    *  The list of know scripts.  Each different script corresponds to the
00245    *  following information:
00246    *
00247    *   - A set of Unicode ranges to test whether the face supports the
00248    *     script.
00249    *
00250    *   - A specific global analyzer that will compute global metrics
00251    *     specific to the script.
00252    *
00253    *   - A specific glyph analyzer that will compute segments and
00254    *     edges for each glyph covered by the script.
00255    *
00256    *   - A specific grid-fitting algorithm that will distort the
00257    *     scaled glyph outline according to the results of the glyph
00258    *     analyzer.
00259    *
00260    *  Note that a given analyzer and/or grid-fitting algorithm can be
00261    *  used by more than one script.
00262    */
